Ce sunt cookie-urile?

Fisierele cookie sunt fisiere text mici trimise de serverul unui site web catre un navigator web (web browser: Chrome, Firefox, Microsoft Edge, Opera), memorie RAM sau unitate de hard disk si apoi stocate acolo. Acestea pot fi utilizate cu scopuri diferite, cum ar fi: personalizarea site-ului web in functie de nevoile unui anumit utilizator, ajutarea unui utilizator sa navigheze pe site-ul web, imbunatatirea calitatii site-ului web pentru utilizator sau stocarea preferintelor utilizatorului si informatiilor de conectare.

Cookie-uri esentiale si neesentiale

Cookie-urile pot fi clasificate drept "esentiale" sau "neesentiale".

Cookie-urile esentiale: prezinta una dintre urmatoarele doua caracteristici:
  • sunt utilizate exclusiv pentru efectuarea sau facilitarea comunicarii intr-o retea;
  • sunt strict necesare pentru furnizarea unui serviciu online (de ex. site-ul nostru sau un serviciu de pe site-ul nostru) solicitat de catre tine.

Cookie-uri neesentiale: Acestea nu se incadreaza in definitia cookie-urilor esentiale deoarece nu sunt strict necesare pentru functionarea unui site sau a serviciilor oferite pe acesta, precum cookie-urile utilizate pentru analizarea comportamentul pe un site web (cookie-uri "analitice") sau cookie-urile utilizate pentru afisarea de reclame.

Poti gasi informatii aditionale despre cookie-uri accesand urmatorul site web ro.wikipedia.org

Site-ul PetProfile.ro utilizeaza 3 cookie-uri esentiale, strict necesare pentru functionarea site-ului si 2 cookie-uri analitice
Cele esentiale sunt:
  • sessionid: un identificator de sesiune sau un jeton de sesiune este o bucata de date care este utilizata in comunicatiile de retea pentru a identifica o sesiune, o serie de schimburi de mesaje conexe;
  • csrftoken: acest cookie este un jeton de securitate ce previne atacuri de tip "Cross-Site Request Forgery" si este esential pentru securitatea site-ului;
  • cookieUse: salveaza consimtamantul utilizatorului referitor la utilizarea cookie-urilor.
Cele analitice sunt:
  • _ga si _ga*: ne ofera date statistice anonimizate despre utilizare site-ului, ne permite sa vedem paginile cele mai accesate si sa putem calcula resursele necesare pentru ca site-ul sa se incarce rapid.
